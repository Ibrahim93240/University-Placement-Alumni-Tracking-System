# University-Placement-Alumni-Tracking-System
A full-stack data engineering project that automates the collection, storage, and analysis of campus placement data.

🛠️ Tech Stack
Database: MySQL (Relational Schema Design)

Language: Python 3.x

Libraries: Pandas (Data Transformation), SQLAlchemy (ORM/Database Bridge)

Tools: MySQL Workbench, Git

Database Architecture
Key Features
Secure Authentication: Implemented URL-encoded credential handling for secure SQL connections.

Automated Ingestion: Python script for transforming raw .csv data into structured SQL records.

Advanced Analytics: SQL-driven insights into average salary packages across different industry tiers.
